Front End
---------

 [ Theme ]
   Benny Mei's "Elephants" theme was used as a base due to it's simplistic
   nature. I then modified it drawing inspiration from Calin Tataru's "Minimal"
   theme and Pieter Robberechts' "Cactus" theme. Note that Cactus is for Hexo,
   not Hugo.

     Cactus    - https://github.com/probberechts/hexo-theme-cactus
     Minimal   - https://github.com/calintat/minimal
     Elephants - https://gitlab.com/meibenny/elephants

 [ Libraries & Frameworks ]

    Font Awesome - https://fontawesome.com/
